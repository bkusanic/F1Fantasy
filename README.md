# 🏎️ F1 Fantasy Predictor

AI-powered web app za Official F1 Fantasy.

## Brzi start

```bash
npm install
cp .env.example .env.local   # dodaj ANTHROPIC_API_KEY
npm run dev                   # → localhost:3000
```

## Deploy (Vercel)
```bash
git init && git add . && git commit -m "init"
# Push na GitHub → import na vercel.com → dodaj ANTHROPIC_API_KEY env var
```

Detalji u README-u projekta.
